<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| domain author -> core constructors | Untrusted or malformed stream, partition, and tenant strings enter typed contracts. |
| kernel crate -> future runtime/storage/adapters | Lower-level deterministic code must not depend on future I/O or protocol crates. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Tampering | `StreamId`, `PartitionKey`, `TenantId` | mitigate | Constructors reject empty strings and tests assert `CoreError::EmptyValue`. |
| T-02-02 | Information Disclosure | `CommandMetadata`, `EventMetadata` | mitigate | Metadata structs include explicit IDs and tenant only; no arbitrary request payload, secrets, or JSON blobs. |
| T-02-03 | Elevation of Privilege | `es-kernel` dependency surface | mitigate | Keep dependencies to `es-core` only and grep/cargo-tree for forbidden runtime/storage/adapter crates. |
</threat_model>
