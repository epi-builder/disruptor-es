| T-05-08 | Elevation of Privilege | domain crate dependency boundary | mitigate | Do not add SQLx, Tokio, runtime, store, adapter, or projection dependencies to `example-commerce`; run `dependency_boundaries` tests. |
| T-05-09 | Tampering | event payload JSON shape | mitigate | Use serde derives on closed Rust enums and value objects; projection code decodes exact variants rather than accepting arbitrary JSON maps. |
</threat_model>
