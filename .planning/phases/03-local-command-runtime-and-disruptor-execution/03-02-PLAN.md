## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| adapter -> CommandGateway | External adapter requests cross into bounded runtime admission. |
| CommandGateway -> shard routing | Tenant and partition fields determine local shard ownership. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-06 | Denial of Service | CommandGateway | mitigate | Use bounded `tokio::sync::mpsc::channel` and `try_send`; map full capacity to `RuntimeError::Overloaded`. |
| T-03-07 | Elevation of Privilege | PartitionRouter | mitigate | Hash `tenant_id.as_str()`, separator byte `0`, and `partition_key.as_str()` so tenants do not share ownership solely by aggregate ID. |
| T-03-08 | Tampering | routing stability | mitigate | Fix `ROUTING_HASH_SEED` and golden route tests for known tenant/key pairs. |
| T-03-09 | Denial of Service | invalid capacity | mitigate | Reject zero ingress capacity and zero shard count with typed errors before channel creation. |
</threat_model>
