## Sampling Rate

- **After every task commit:** Run the task-local `<verify><automated>` command from the active PLAN.md. Long-running Criterion/live HTTP commands in Plan 13.1-06 are phase-gate evidence checks; the task-local guard for those tasks uses faster compile/smoke validation first.
- **After every plan wave:** Run `cargo test -p es-runtime runtime_flow -- --nocapture && cargo test -p app http_stress -- --nocapture`.
- **Before `/gsd-verify-work`:** Full suite must be green
- **Max feedback latency:** 180 seconds for task-local checks, excluding explicit live HTTP or Criterion comparison runs
